# Threads & Concurrency Educational Dashboard

An interactive dashboard designed to visualize and explain complex threading and concurrency concepts. This project features dynamic 3D visualizations and real-time examples to help students understand parallel processing.

## ‚ú® Key Features
- **3D CPU Architecture**: Interactive 3D visualization of multi-core processors showing thread execution in real-time.
- **Thread Lifecycle Management**: Visual guide to understanding thread states (New, Runnable, Running, Blocked, Terminated).
- **Concurrency Simulations**: Live demos of common concurrency problems like Race Conditions and Deadlocks.
- **Synchronization Primitives**: Interactive modules explaining Mutexes, Semaphores, and Monitors.
- **Performance Analytics**: Real-time metrics dashboard for CPU usage, context switches, and throughput.
- **Interactive Controls**: Users can pause, resume, and adjust thread counts to see how the system reacts.
